### APIs Automated (6 endpoints)
1. **POST** `/token` - OAuth2 authentication with token validation
2. **POST** `/addPassenger` - Create passenger with form-urlencoded data
3. **GET** `/viewPassengerById/{id}` - Single passenger retrieval + schema validation
4. **GET** `/viewPassengerList` - Array response validation with schema
5. **GET** `/viewPassengerByNameMobile/{name}/{mobile}` - Search functionality
6. **DELETE** `/deletePassengerById/{id}` - Delete operation with verification

**Validations:** Status codes (200), JSON schema compliance, response time, data integrity
